Retail Sales Data Analysis 📊

Project Overview

This project analyzes a small retail dataset to understand sales patterns and customer behavior. The analysis is performed using Python and focuses on identifying key insights from sales data.

Technologies Used

- Python
- Pandas
- Seaborn
- Matplotlib

Features

- Data inspection and cleaning
- Descriptive statistics of sales data
- Sales analysis by category
- Spending distribution by gender
- Time series visualization of sales trends

Dataset

The dataset used in this project contains the following columns:

- Date – Date of the transaction
- Category – Product category (Electronics, Clothing, Home)
- Sales – Amount of sales
- Gender – Customer gender

Visualizations

The project includes:

- Bar chart for Sales by Category
- Box plot for Spending Distribution by Gender
- Line chart showing Sales Trend Over Time

Key Insights

- Electronics category generated the highest sales.
- Spending patterns differ between male and female customers.
- Sales trends can be observed over time using time-series visualization.

How to Run the Project

1. Install required libraries:

pip install pandas seaborn matplotlib

2. Run the Python script:

python retail_sales_analysis.py

Recommendations

- Focus marketing strategies on high-performing categories like Electronics.
- Further analyze customer behavior to understand spending differences.
